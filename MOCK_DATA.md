## How to Test

**Option 1: Paste one by one**

2. Select Type (Bug or Feature)
3. Select Customer Tier (mix of Free/Pro/Enterprise)
4. Paste the text
5. Click Submit

**Option 2: Bulk test**
1. Add 5-10 items
2. Go to project page
3. Click "Analyze"
